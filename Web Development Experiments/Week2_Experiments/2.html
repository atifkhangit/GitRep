Webkit-Transition Property

box 1

On mouse hover above box slide right using the webkit-transition property of CSS

box 2

On mouse hover above box slide left using the webkit-transition property of CSS

box 3

On mouse hover above box rotates at 50 degree using the webkit-transition property of CSS

box 4

Webkit Transition:

The webkit transition property in css gives the power to animate the objects on HTML page.
For above animation a container is used and the property of container such as scale,rotate,sliderright and sliderleft have been used.
