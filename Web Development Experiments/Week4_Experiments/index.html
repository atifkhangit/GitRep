Text file data display using AJAX:

The purpose of this experiment is to read a text file from server and display its data on button click.
When the display data button is clicked the API search for text file (TextFile.txt) present on the server and reads it and displays the data from the text file.
When the display error button is clicked the API search for text file (TextFile1.txt) but as the file is not present on server it displays error.
