Using Flickr API for json callback

Fetch Northeastern University Images from Flickr!

Code Snippet:

var fetchAPI = 'http://api.flickr.com/services/feeds/photos_public.gne?jsoncallback=?';
$.getJSON(fetchAPI, {
tags: 'Northeastern University',
tagmode: 'any',
format: 'json'
},
onSuccess);

Displaying images using Flickr API :

The purpose of this experiment is to display images with jSon callback using flickr api.
The API search images for tag parameter with values Northeastern University.
The cache response of AJAX has been disabled so that the images are not cached and everytime the API is accessed a different random
set of images with tags "Northeaster University" are displayed
