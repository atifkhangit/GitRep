AngularJS Controller to add data dynamically on webpage

Code Snippet:

HTML:

<li ng-repeat="email in emails" class="ng-binding"> {{email}}</li>

JS:

$scope.emails.push($scope.newemail);
$scope.newemail = "";

Description:

In this experiment, once the add button is clicked, the AngularJS initializes the controller "EmailController" and creates and assign the $scope variable to the function that add the email id on the HTML page.
The ng-scope attribute defines the scope of AngularJS in div of controller and ng-binding tells Angular to replace the text content of the specified "li" element with the value of a email array.
The ng-repeat attribute loops through the array of emails and return the email id's to the controller via $scope which displays it on this page.
