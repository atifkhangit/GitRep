Description:

In this experiment, jQuery has been used to perform in place editing. When the text in the editable DIV is clicked, a text box appears. The data entered in the new text box replaces the old data in the tag <p>. On click, the current <p> data hides and a new textbox appears at the same place. The new textbox after editing is then focussed and displayed on screen.
The jQuery code then checks whether the initial text has been changed or not. If not will display the same text.
If changed will display the updated text.
