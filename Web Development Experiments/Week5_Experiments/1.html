Basic Angular JS directives

Display model data on screen:

Please enter your name:

Hi {{name1}}

Hi {{ name1 }}

Description:

Anything entered in the text box is displayed beside "Hi".
In the first line, the text "Hi" is hard coded and only the data entered in text field appears beside "Hi"
In the second line, "Hi" has been set to appear along with the data entered in text field.

AngularJS filter to change case of data:

Please enter your name:

Hi {{ name2 }}

Name in lower case: {{ name2 | lowercase }}

Name in upper case: {{ name2 | uppercase }}

Description:

The code uses the filter attribute from AngularJS library. Usually the filter attribute is used to filter the data
Here I have used the filter to convert data from lower to upper and upper to lower case
The filter works without any additional javascript.
