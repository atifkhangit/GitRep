Google map with controls

JS:

var mapFeatures = {
center: latlon,
zoom: 13,
mapTypeControl: true,
mapTypeId: google.maps.MapTypeId.ROADMAP
};
getMap = new google.maps.Map(
document.getElementById("mapspace"), mapFeatures
);

Description:

In this experiment,The Google map API with controls has been used. The javascript function getLocation() finds the current location coordinates and pass it to map specification function mapFeatures(). The mapFeatures() function contains the specific attributes of google map API such as mapType,zoom level and other navigation controls. The api marker called google.maps.Marker takes the coordiantes and pinpoint the location the map.
