Simple Static Google Map

Code Snippet:

var coord = loc.coords.latitude + "," + loc.coords.longitude;
var mp = "http://maps.googleapis.com/maps/api/staticmap?center="+ coord + "&zoom=16&size=400x400&sensor=false";

Description:

In this experiment,The static map API of google maps has been used. The API takes the current location and display a static map of the region with respect to the coordinates obtained while getting the current location. When the button is clicked the API gets the current location generate a static map and the map is displayed in the image tag.
