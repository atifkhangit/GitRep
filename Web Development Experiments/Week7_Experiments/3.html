Get Geocode for the address






Code Snippet:

marker = new google.maps.Marker(
{
map: map,
position: results[0].geometry.location,
draggable: true
});
document.getElementById('lat').value = marker.getPosition().lat();
document.getElementById('lon').value = marker.getPosition().lng();

Description:

In this experiment,The Google maps javascript API v3 has been used. The geocoding service of Google map API has been used to get the latitude and longitude of the address entered in the text box. Also the location entered in the text box is pinpointed on the map.
