Load data from another HTML file using XMLHTTPRequest object

JS:

request.open("GET", url, false);
request.send();

Description:

In this experiment, The XMLHTTPRequest is used to load data from another HTML file. The javascript make a synchnronous call to file on server.
The server opens the html file and returns the data in form of text. The javascript then renders the data in the Div tag with id "displaydata"
The request.send() method sends the second HTML file to server and get the data of second HTML file for rendering using javascript on this page.
