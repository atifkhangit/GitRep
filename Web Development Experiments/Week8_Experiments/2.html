Sine wave on Canvas using jQuery

The main formula for generating sine wave is in below code snippet.

Code Snippet:

for (x = -1; x <= cnv.width; x++)
{
y = (Math.random() * 2 + Math.cos(t + x * 0.04) * 50);
context.lineTo(x, cnv.height * 0.6 - y);
}

Description:

In this experiment,A Sine wave has been generated on canvas using jQuery. The sine wave is generated using the above formula to generate lines which re displayed at regular intervals. The setInterval() method has been used to set the intervals.
