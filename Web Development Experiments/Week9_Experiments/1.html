Description:

In this experiment, I have used internet radio API Dirble. The response format is XML returning three fields: Category id, Category and Description.The description field in xml response has many empty nodes which was causing the jquery to fail. The empty nodes have been handled by replacing the empty node data with a "-'" in jQuery.
