Get all stations for Rock category


Description:

This is an extension of the previous experiment. The category ID obtained from previous experiment has been used to get all the stations listed for Rock Category. For simplicity, only 10 stations have been displayed. Clicking the 'Get station' button was appending the data from subsequent API call in the existing DOM tree. I resolved this by disabling the 'Get Station' on first click and re-enabling when reset is clicked.
