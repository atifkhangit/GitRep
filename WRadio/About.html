What is WRadio:

WRadio (Web Radio) is a webcasting site. The site provide access to internet radio stations broadcasting across the world in different languages. The home page displays set of random stations for any Genre from Classical, Rock, Jazz to country music and live broadcasting. You can select a number of stations based on the Genre. The data source for the streaming of radio station is "Dirble API" .
